Mallary CLI is the official command-line interface for [mallary.ai](https://mallary.ai).

It gives developers, operators, scripts, CI jobs, and AI agents a fast way to work with the same public platform that powers the Mallary dashboard, REST API, and MCP server.

## What You Can Do

With the CLI you can:

- upload local media files to Mallary
- create and schedule posts
- inspect jobs and grouped posts
- fetch post analytics
- manage webhooks
- manage brand settings
- disconnect platforms

## API Model

Mallary CLI is a direct client for the public Mallary API.

It does not bypass plan limits, feature gates, or platform rules.

## Plan Access

CLI access is available on paid plans only:

- Starter
- Pro
- Business
