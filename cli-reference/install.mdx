---
title: "Install"
description: "Install, update, and remove the Mallary CLI."
---

 View the package on NPM: https://www.npmjs.com/package/@mallary/cli

## npm

```bash
npm install -g @mallary/cli
```

## npx

```bash
npx @mallary/cli --help
```

## Update

```bash
npm install -g @mallary/cli@latest
```

## Uninstall

```bash
npm uninstall -g @mallary/cli
```
