---
title: "Authentication"
description: "Authenticate Mallary CLI with your Mallary API key."
---

Mallary CLI uses environment-variable auth only.

Get your API key from [mallary.ai](https://mallary.ai).

## Required Environment Variable

```bash
export MALLARY_API_KEY="your_mallary_api_key"
```

## Plan Requirement

The CLI is available on paid plans only:

- Starter
- Pro
- Business
