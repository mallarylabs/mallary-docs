## 2. Choose your integration path

Mallary supports multiple integration layers so you can work at the level you prefer.
Use the API for full control, MCP for higher-level automation, or the CLI for quick scripting and local workflows. All options map to the same underlying system, so you can switch or combine them as needed.

<Columns cols={1}>
  <Card title="API" icon="terminal" href="/api-reference/introduction">
    Build your own integration. Simple REST API with endpoints for posting,
    scheduling, media uploads, and analytics. Works with any language or
    framework.
  </Card>
  <Card
    title="MCP"
    icon="message-square-code"
    href="/mcp-reference/introduction"
  >
    Native Model Context Protocol integration. Your AI assistant can schedule
    posts, check analytics, and manage accounts — all from the chat.
  </Card>
  <Card title="CLI" icon="package" href="/cli-reference/introduction">
    Give your OpenClaw agent social media superpowers with the Mallary.ai CLI.
    It can plan, write, and post content autonomously — across 10 platforms from
    a single chat message.
  </Card>
</Columns>
