---
title: "Authentication"
description: "Authenticate to the Mallary MCP server with your Mallary API key."
---

Mallary MCP uses the same API key model as the REST API.

## Header

Send your Mallary API key as a bearer token:

```http
Authorization: Bearer <mallary_api_key>
```

## Identity Resolution

Identity is always resolved from the API key owner. You do not separately pass a user ID when calling MCP tools.

## Base URL

Mallary MCP is served from:

```text
https://mallary.ai
```

## Endpoint

Use the MCP server at:

```text
/mcp
```
