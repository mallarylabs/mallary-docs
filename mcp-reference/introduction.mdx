Mallary exposes a remote MCP server over Streamable HTTP for public API parity.

## Base URL

```text
https://mallary.ai
```

## MCP Endpoint

Mallary serves MCP at:

```text
/mcp
```

Supported methods:

- `POST /mcp`
- `GET /mcp`
- `DELETE /mcp`

## What You Can Do

Mallary MCP gives MCP clients access to the same publishing surface exposed by the public Mallary API, including posting, uploads, analytics, platform disconnects, and webhook management.
